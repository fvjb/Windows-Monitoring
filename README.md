## WINDOWS MONITORING
### Windows AppLocker
Windows Environments often run many applications that we are not aware of. This might represent a significant risk to IT. To allow a higher level of control on what applications run on a Windows System, AppLocker was introduced about 15 years ago. 

Most Administrators hesitate to start using it in production as so many projects happened to fail ([enjoy this article from 11 years ago](https://www.itninja.com/blog/view/the-right-wrong-ways-to-prevent-inappropriate-app-execution))
